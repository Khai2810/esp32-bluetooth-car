# BluetoothCar Project

## Overview
The BluetoothCar project is designed for the ESP32 microcontroller, enabling Bluetooth functionality to control a car remotely. This project serves as a foundation for building a Bluetooth-controlled car, allowing for further enhancements and customizations.

## Project Structure
```
BluetoothCar
├── platformio.ini
├── src
│   └── main.cpp
├── lib
├── include
├── test
└── README.md
```

## Setup Instructions
1. **Install PlatformIO**: Ensure you have PlatformIO installed in your development environment.
2. **Clone the Repository**: Clone this repository to your local machine.
3. **Open the Project**: Open the `BluetoothCar` folder in PlatformIO.
4. **Install Dependencies**: Run the command to install any required libraries specified in `platformio.ini`.

## Usage
- Connect the ESP32 to your computer and upload the code from `src/main.cpp`.
- Use a Bluetooth-enabled device (like a smartphone) to connect to the ESP32.
- Control the car using the provided Bluetooth commands.
